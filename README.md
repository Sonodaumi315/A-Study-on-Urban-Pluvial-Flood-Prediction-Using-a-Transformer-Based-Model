This repository stored the codes utilized in our study, Liu, Z., Zhu, D., Luo, J., & Li, D. (2025). Urban Pluvial Flood Maximum Water Depth Prediction Using a Transformer-CNN Hybrid Deep Learning Model with Data Augmentation.

The proposed data augmentation method is presented in dataset.py.

The three utilized models are presented in model_unet.py, model_vit.py, and model_mobilevitv2.py.
